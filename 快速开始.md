# 命运轮盘 Web版 - 快速开始指南

## 🎮 游戏已成功创建！

服务器正在运行中: **http://localhost:3000**

## 📋 如何开始游戏

### 方法1: 单机对战（与AI对战）
1. 打开浏览器访问 http://localhost:3000
2. 点击"开始游戏"
3. 选择"单机对战"
4. 选择AI难度（困难/专家/地狱）
5. 开始游戏！

### 方法2: 多人联机
1. **创建房间**:
   - 点击"开始游戏" → "多人联机" → "创建房间"
   - 输入你的名字
   - 获得房间号，分享给好友

2. **加入房间**:
   - 点击"开始游戏" → "多人联机" → "加入房间"
   - 输入房间号和你的名字
   - 等待游戏开始

3. **快速匹配**:
   - 点击"开始游戏" → "多人联机" → "快速匹配"
   - 系统自动匹配对手

## 🎯 游戏操作

### 回合流程
1. **使用灵物**: 点击你的灵物卡牌使用（可选）
2. **使用命运卡牌**: 点击"使用命运卡牌"按钮结束回合
3. **选择目标**: 选择对自己还是对手使用

### 获胜条件
- 将对手的生命值降至0或以下

## 🎨 界面说明

### 对手区域（顶部）
- 显示对手的生命值
- 显示对手的灵物（隐藏灵物显示为"神秘护符"）
- 显示对手的状态效果

### 中央区域
- 命运牌堆剩余数量
- 回合指示器
- 游戏日志

### 玩家区域（底部）
- 你的生命值
- 你的灵物卡牌（点击使用）
- 你的状态效果

## 💡 游戏提示

1. **护身符和镜子**是隐藏灵物，对手看不到具体是哪个
2. 失去生命时会获得**2个灵物**作为补偿
3. 某些卡牌可以获得**额外回合**
4. 合理使用灵物组合可以创造强大的连招
